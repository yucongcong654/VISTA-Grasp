## 签署方式说明

本项目基于 **Apache-2.0** 许可证开源。为了最大限度降低外部开发者的参与门槛，我们使用了极简的开发者来源证书（DCO）形式来替代复杂的线下签名或 PDF 回传文件流程。

**您只需要在每次执行 Git Commit 时，在提交描述信息的最后附带一行 `Signed-off-by` 签名，即表示您已阅读、理解并同意本 CLA 的全部条款。**

您可以通过给 `git commit` 加上 `-s` 参数来让 Git 为您自动生成这一行：

```bash
git commit -s -m "feat: add support for new robot arm kinematics"
```

您的 commit 信息将会像这样：

```text
feat: add support for new robot arm kinematics

Signed-off-by: Your Name <your.email@example.com>
```
